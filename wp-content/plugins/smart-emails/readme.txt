== Description ==

The Customizer allows you to preview live changes to your WooCommere Emails. You can select different WooCommerce Email Templates and Styles to preview and customize.

== Installation ==

1. Ensure you have latest version of [WooCommerce](https://www.storeapps.org/woocommerce/) plugin installed
2. Unzip and upload the folder 'smart-emails' to your `/wp-content/plugins/` directory
3. Activate 'Email Customizer For WooCommerce (Smart Emails)' through the 'Plugins' menu in WordPress
